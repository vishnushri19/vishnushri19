### 🌐 Dynamic Traffic Decision Engine
**[`dtde-lab`](https://github.com/vishnushri19/dtde-lab)**

A composite-scoring engine for multi-cloud / multi-region traffic routing — health, latency, and security/policy signals fused into a single live score per backend, with security treated as a **hard constraint**, not a weighted preference.

```
S_i = (w_h · H_i) + (w_l · 1/(L_i+1)) + (w_s · S_i)
```

`Region-A fails → score collapses → Region-B takes over`

**Roadmap:** LSTM-based predictive failover · gossip-protocol consensus across DTDE nodes

</td>
<td width="50%" valign="top">

### 🔍 LLM SAST Triage
**[`llm-sast-triage`](https://github.com/vishnushri19/llm-sast-triage)**

Scanner-first triage pipeline: Semgrep finds it, deterministic logic clusters it, a local Ollama model explains it — evidence is never dropped or invented in the process.

| Metric | Result |
|---|---|
| Recall | 1.00 |
| Precision | 0.78 |
| Severity accuracy | 1.00 |
| False-positive triage accuracy | 1.00 |

**Latest:** `v0.3.0` — Phase 3, 10-target evaluation
